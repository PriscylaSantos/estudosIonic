## Estudos sobre o Ionic2 e Ionic3

### Comandos

Iniciando um app Ionic2 

    $ ionic start nomeApp template --v2 

Instalando depedências  

    $ cd nomeApp/
    $ npm install

Visualizar app no browser

    $ ionic serve   
    ou 
    $ ionic serve --lab

Gerar paǵinas automaticamente

    $ ionic g page nomePagina
